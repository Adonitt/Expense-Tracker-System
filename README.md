# Income And Expense Backend

This is a Spring Boot backend application for managing income and expenses.  

This backend application is built with Spring Boot for managing personal or business finances. It allows users to track their income and expenses efficiently, providing CRUD operations for users, secure authentication with JWT, password encryption with BCrypt, and role management. The application uses PostgreSQL as the database and Flyway for database migrations.

It provides:
- User registration and login with JWT authentication
- Password encryption using BCrypt
- CRUD operations for users (create, read, update, delete)
- Role management (USER, ADMIN)
- PostgreSQL integration with Flyway migrations
- Exception handling for invalid data and duplicate emails

Built with Java 17, Spring Boot.
